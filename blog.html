Important FAQ

What is the Document Object Model (DOM)?

The DOM is a representation of a web page's structure, turning the HTML into a tree of objects that can be accessed and modified using JavaScript.

How do you select an element from the DOM?

An element can be selected using methods like querySelector(), querySelectorAll(), getElementById(), getElementsByClassName() and getElementsByTagName().

What is event delegation in the context of the DOM, and why is it useful?

Event delegation involves placing a single event listener on a parent element to handle events for its child elements. This is useful for managing dynamic content and reducing the number of event listeners in the document.

How do you manipulate an element's attributes and styles using the DOM?

Attributes can be manipulated with setAttribute() and getAttribute(). For styles, the style property can be used, such as element.style.color = "gray".
